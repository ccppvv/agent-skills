## 产品经理

### 核心职责
- 需求调研与分析
- 产品规划与设计
- 需求文档(PRD)撰写
- 跨部门沟通协调
- 产品迭代与优化

### 典型工作流程
```
需求收集 → 需求分析 → 方案设计 → PRD撰写 → 评审沟通 →
开发跟进 → 测试验收 → 上线复盘 → 数据分析 → 迭代优化
```

### 高频任务清单

| 任务 | 频次 | 典型耗时 | 痛点 |
|------|------|---------|------|
| 用户需求调研 | 2-3次/周 | 4-6h | 整理访谈记录费时费力 |
| PRD文档撰写 | 1-2次/周 | 6-8h | 格式规范、描述清晰度把控难 |
| 竞品分析 | 1次/周 | 3-4h | 信息搜集分散，难以系统化 |
| 需求评审准备 | 2-3次/周 | 2-3h | PPT制作占用大量时间 |
| 数据分析报告 | 1次/周 | 3-4h | 数据提取和图表制作繁琐 |
| 跨部门邮件沟通 | 每天 | 1-2h | 邮件措辞需反复推敲 |

### AI应用潜力场景

#### 🔥 高潜力场景 (Level 2-3)

**1. PRD文档生成**
- **当前痛点**：格式不统一、描述不够清晰、遗漏边界case
- **AI解决方案**：
  - 使用Claude基于需求要点生成结构化PRD
  - Prompt模板：见 `prompt-templates.md` → "PRD生成"
- **预期提效**：60-70% (AI生成框架+人工补充细节)
- **工具推荐**：Claude (长文档能力强)

**2. 用户访谈整理**
- **当前痛点**：录音转文字+提炼关键信息费时
- **AI解决方案**：
  - 使用讯飞听见/剪映转录音为文字
  - Claude提取核心需求和用户痛点
- **预期提效**：70-80%
- **工具推荐**：讯飞听见 + Claude

**3. 竞品分析报告**
- **当前痛点**：信息搜集分散，难以系统对比
- **AI解决方案**：
  - Perplexity搜索竞品功能信息
  - Claude生成对比分析表和SWOT分析
- **预期提效**：50-60%
- **工具推荐**：Perplexity + Claude

#### 🟡 中等潜力场景 (Level 1-2)

**4. 需求评审PPT**
- **AI解决方案**：Claude生成演讲稿大纲 + Gamma AI生成PPT
- **预期提效**：40-50%

**5. 数据分析解读**
- **AI解决方案**：ChatGPT Code Interpreter分析Excel数据 + 生成洞察
- **预期提效**：40-50%

**6. 邮件润色与回复**
- **AI解决方案**：Claude/GPT优化邮件措辞，确保专业性
- **预期提效**：30-40%

### Prompt模板示例

**需求访谈整理：**
```
请分析以下用户访谈记录，提炼出：
1. 核心需求（用户真正想解决的问题）
2. 痛点场景（具体在什么情况下遇到问题）
3. 期望解决方案（用户提到的期望功能）
4. 潜在需求（用户未明确提及但可能需要的功能）

访谈记录：
[粘贴转录文本]
```

##运营

### 核心职责
- 用户增长与留存
- 内容运营与创作
- 活动策划与执行
- 数据监测与优化
- 用户反馈处理

### 典型工作流程
```
目标制定 → 策略规划 → 内容生产 → 渠道投放 →
效果监测 → 数据分析 → 优化调整 → 复盘总结
```

### 高频任务清单

| 任务 | 频次 | 典型耗时 | 痛点 |
|------|------|---------|------|
| 公众号/短视频内容创作 | 3-5次/周 | 2-4h/篇 | 选题困难、内容质量不稳定 |
| 活动策划方案 | 1-2次/周 | 4-6h | 创意枯竭、细节考虑不周 |
| 用户反馈处理 | 每天 | 1-2h | 重复性问题回复费时 |
| 数据日报/周报 | 每天/周 | 1-2h | 数据提取和报表制作 |
| 竞品动态监测 | 2-3次/周 | 1-2h | 信息分散，难以持续追踪 |
| 投放文案撰写 | 5-10次/周 | 30min-1h | A/B测试变体生成耗时 |

### AI应用潜力场景

#### 🔥 高潜力场景

**1. 内容创作辅助**
- **应用领域**：公众号文章、短视频脚本、产品文案
- **AI解决方案**：
  - Claude生成文章大纲和初稿
  - GPT-4生成多种文案变体用于A/B测试
  - 剪映智能文案生成短视频脚本
- **预期提效**：50-70%
- **Prompt模板**：见 `prompt-templates.md` → "内容创作"

**2. 用户反馈智能分类与回复**
- **当前痛点**：每天处理大量重复性用户问题
- **AI解决方案**：
  - 搭建AI客服（Dify/Coze）处理常见问题
  - Claude生成个性化回复模板
- **预期提效**：60-80%

**3. 活动策划方案生成**
- **AI解决方案**：
  - 输入活动目标、预算、目标用户
  - Claude生成完整策划案框架
  - 包含：活动主题、玩法设计、传播策略、预算分配
- **预期提效**：40-60%

#### 🟡 中等潜力场景

**4. 数据报表自动生成**
- **AI解决方案**：Excel/Google Sheets + AI插件 或 Tableau AI
- **预期提效**：50-70%

**5. 竞品动态监测**
- **AI解决方案**：设置Google Alerts + Perplexity定期搜索 + Claude总结
- **预期提效**：40-50%

## 开发工程师

### 核心职责
- 功能开发与实现
- Bug修复与优化
- 代码审查(Code Review)
- 技术文档编写
- 系统维护与升级

### 典型工作流程
```
需求理解 → 技术方案设计 → 代码实现 → 单元测试 →
Code Review → 联调测试 → 上线部署 → 问题修复 → 文档更新
```

### 高频任务清单

| 任务 | 频次 | 典型耗时 | 痛点 |
|------|------|---------|------|
| 代码编写 | 每天 | 4-6h | 重复性代码、样板代码占比高 |
| Bug调试 | 每天 | 1-3h | 定位问题耗时，缺乏调试思路 |
| Code Review | 2-3次/周 | 1-2h | 代码量大，关注点容易遗漏 |
| 技术文档编写 | 1-2次/周 | 2-3h | 文档更新不及时，格式不统一 |
| 日志分析 | 每天 | 30min-1h | 海量日志中筛选有效信息 |
| 接口文档编写 | 1次/周 | 1-2h | 手动维护Swagger/API文档 |

### AI应用潜力场景

#### 🔥 高潜力场景

**1. 代码补全与生成**
- **当前痛点**：重复性CRUD代码、样板代码占开发时间30-40%
- **AI解决方案**：
  - GitHub Copilot：实时代码补全
  - Claude Code/Cursor：基于需求生成完整功能模块
- **预期提效**：40-60%
- **工具推荐**：GitHub Copilot + Cursor

**2. Bug调试辅助**
- **AI解决方案**：
  - 将错误日志/堆栈信息输入Claude
  - Claude分析可能原因并提供修复建议
  - Claude Code直接修复代码
- **预期提效**：30-50%

**3. Code Review自动化**
- **AI解决方案**：
  - 使用Claude/GPT分析PR代码
  - 检查：潜在bug、性能问题、代码规范、安全漏洞
- **预期提效**：40-50%
- **Prompt模板**：见 `prompt-templates.md` → "代码审查"

**4. 技术文档自动生成**
- **AI解决方案**：
  - Claude基于代码生成函数说明、API文档
  - 自动更新README和使用示例
- **预期提效**：60-70%

#### 🟡 中等潜力场景

**5. 日志分析与异常检测**
- **AI解决方案**：Claude分析日志模式，识别异常
- **预期提效**：40-50%

**6. 单元测试生成**
- **AI解决方案**：Copilot/Claude生成测试用例
- **预期提效**：50-60%

### Prompt模板示例

**Bug调试：**
```
我遇到了以下错误，请帮我分析可能的原因并提供修复建议：

错误信息：
[粘贴错误堆栈]

相关代码：
[粘贴代码片段]

运行环境：
- 语言/框架：[如 Python 3.10 / Django 4.2]
- 操作系统：[如 macOS 13]
- 其他信息：[...]

请提供：
1. 错误原因分析
2. 具体修复步骤
3. 预防类似问题的建议
```

## HRBP

### 核心职责
- 人才招聘与配置
- 员工关系管理
- 绩效管理与辅导
- 组织发展与培训
- 薪酬福利优化

### 典型工作流程
```
人才盘点 → 招聘需求分析 → 候选人筛选 → 面试评估 →
Offer谈判 → 入职培训 → 绩效辅导 → 员工关怀 → 离职面谈
```

### 高频任务清单

| 任务 | 频次 | 典型耗时 | 痛点 |
|------|------|---------|------|
| 简历筛选 | 每天 | 1-2h | 海量简历，难以快速识别匹配度 |
| 面试评估报告 | 3-5次/周 | 30min/次 | 格式不统一，关键信息提炼难 |
| 员工满意度调研 | 1次/季度 | 4-6h | 问卷设计和结果分析耗时 |
| 培训课程设计 | 2-3次/月 | 4-8h | 课程内容开发工作量大 |
| 绩效反馈沟通 | 每月 | 2-3h | 反馈措辞需要精心打磨 |
| 组织诊断报告 | 1次/季度 | 8-12h | 数据整合和洞察提炼复杂 |

### AI应用潜力场景

#### 🔥 高潜力场景

**1. 简历智能筛选与评估**
- **当前痛点**：每日简历量大，人工筛选效率低
- **AI解决方案**：
  - Claude分析简历与JD匹配度
  - 自动提取关键信息：技能匹配、项目经验、稳定性
  - 生成筛选理由和面试建议
- **预期提效**：60-70%

**2. 面试评估报告生成**
- **AI解决方案**：
  - 输入面试记录/评估表
  - Claude生成结构化评估报告
  - 包含：能力评估、优劣势分析、录用建议
- **预期提效**：50-60%

**3. 员工调研问卷设计**
- **AI解决方案**：
  - 输入调研目标（如：识别团队协作问题）
  - Claude生成问卷题目和量表设计
  - 自动分析问卷结果并生成洞察
- **预期提效**：40-60%

**4. 培训课程内容生成**
- **AI解决方案**：
  - 基于培训目标和受众特征
  - Claude生成课程大纲、PPT内容、案例素材
- **预期提效**：50-70%

#### 🟡 中等潜力场景

**5. 绩效反馈措辞优化**
- **AI解决方案**：Claude润色反馈语言，确保建设性和专业性
- **预期提效**：30-40%

**6. 组织诊断报告**
- **AI解决方案**：基于人才盘点数据，Claude生成分析报告
- **预期提效**：40-50%

## 项目经理 (PM)

### 核心职责
- 项目计划与排期
- 资源协调与分配
- 风险识别与管理
- 进度跟踪与汇报
- 跨团队沟通协调

### 典型工作流程
```
项目启动 → 需求确认 → 计划制定 → 资源分配 →
执行监控 → 风险应对 → 进度汇报 → 项目验收 → 复盘总结
```

### 高频任务清单

| 任务 | 频次 | 典型耗时 | 痛点 |
|------|------|---------|------|
| 项目计划制定 | 每个项目1次 | 4-8h | WBS分解、依赖关系梳理复杂 |
| 周报/日报撰写 | 每天/周 | 30min-1h | 信息收集分散，格式化费时 |
| 会议纪要整理 | 每天 | 30min-1h | 手动记录容易遗漏重点 |
| 风险识别与应对 | 1次/周 | 1-2h | 缺乏系统化方法论 |
| 项目复盘报告 | 每个项目结束 | 3-4h | 经验教训提炼不够深入 |
| 跨部门邮件沟通 | 每天 | 1-2h | 信息同步效率低 |

### AI应用潜力场景

#### 🔥 高潜力场景

**1. 项目计划生成**
- **当前痛点**：WBS分解、工期估算、依赖关系梳理费时
- **AI解决方案**：
  - 输入项目目标、交付物、约束条件

- **预期提效**：40-60%

**2. 会议纪要自动生成**
- **AI解决方案**：
  - 使用讯飞听见/Otter.ai转录会议录音
  - Claude提炼决策事项、行动项、责任人
  - 自动生成结构化会议纪要
- **预期提效**：70-80%

**3. 周报/日报自动生成**
- **AI解决方案**：
  - 整合：Jira任务进度、Git提交记录、会议纪要
  - Claude生成项目进展报告
- **预期提效**：60-70%

**4. 风险识别与应对方案**
- **AI解决方案**：
  - Claude基于项目信息识别潜在风险
  - 生成风险矩阵和应对策略
- **预期提效**：40-50%

#### 🟡 中等潜力场景

**5. 项目复盘报告**
- **AI解决方案**：基于项目数据和反馈，Claude生成复盘报告
- **预期提效**：40-50%
