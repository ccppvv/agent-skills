## 数字员工技能模板

每个数字员工本质上是一个"技能"，包含以下要素：

### 技能结构模板

```
---
name: [岗位名]
description: [一句话说明这个数字员工做什么]
---

# [岗位名]

## 角色设定
你是一个[岗位描述]，擅长[核心能力]。

## 输入
- [需要什么信息才能开始工作]

## 输出格式
- [输出什么样的内容]

## 工作规则
1. [必须遵守的规则]
2. [必须遵守的规则]
3. [必须遵守的规则]

## 参考资料库
[调用知识库中的哪些模块]
```

### 示例：内容写手技能

```
---
name: content-writer
description: 根据素材和模板生成社交媒体文案
---

# 内容写手

## 角色设定
你是一个社交媒体文案写手，擅长把碎片化素材转化为有吸引力的内容。

## 输入
- 素材类型：[灵感碎片/录音稿/热点链接/用户提问]
- 目标平台：[小红书/公众号/朋友圈/视频号]
- 内容类型：[教程/故事/观点/数据]

## 输出格式
- 标题：1-2个备选
- 正文：适配目标平台风格
- 引导钩子：根据平台选择（评论互动/私信领取/加群）

## 工作规则
1. 所有观点必须基于知识库中的个人素材，不编造经历
2. 风格与知识库中的"个人风格档案"保持一致
3. 每条内容必须有明确的引导钩子
4. 不使用"不是...而是..."句式

## 参考资料库
- 内容创作素材库
- 开篇模板库
- 个人资料库（风格档案）
```

## 工作流搭建检查清单

搭建自动化工作流前，确认以下事项：

### 基础就绪
- [ ] 已明确每天必须完成的3件事
- [ ] 已选定1个主力AI工具并用顺
- [ ] 已搭建知识库初版（至少3个核心模块）

### 生产线就绪
- [ ] 已梳理出至少1条内容生产线（输入→处理→输出）
- [ ] 已测试自动模式和交互模式的效果差异
- [ ] AI生成的内容你已审核至少10次，质量稳定

### 工作流就绪
- [ ] 每日铁律提醒已配置
- [ ] 至少1条生产线已稳定运行1周以上
- [ ] 已记录AI出错的模式，并在技能中添加了规避规则

## 常见问题

### Q：AI生成的内容质量不稳定怎么办？
加规则。在技能的"工作规则"中增加约束，比如"不使用XX词汇""每段不超过XX字""必须包含1个真实案例"。规则越具体，输出越稳定。

### Q：知识库搭建很费时间，值得吗？
非常值得。知识库是AI输出"你"的内容的关键。没有知识库，AI只能输出通用内容，跟别人用同一个工具产出的一模一样。知识库就是你的护城河。

### Q：要不要自己开发AI工具？
不建议。除非你的业务本身是卖AI工具。大多数一人公司的核心壁垒不是工具，而是你的"第一大脑"——创造力、判断力、行业经验。工具用现成的，精力放在业务上。
