# 路由规则

## 入口识别

- 提到 `manage`、`后台页面`、`React`、`styled-components`：走 manage。
- 提到 `agent-flow`、`节点编辑器`、`Vue`、`DESIGN-TOKENS`：走 agent-flow。
- 同时提到两边：先改 token 源头，再改消费侧。

## 文件锚点

- manage：`packages/apps/manage/src/views/**`、`packages/apps/manage/src/components/**`
- agent-flow：`packages/apps/agent-flow/src/components/**`、`packages/apps/agent-flow/src/flow/**`
