# Sunset Synthwave

Synthwave sunset style with purple-orange gradients creating a retro-futuristic feel.

## Color Palette

- **Sunset Violet**: `#2d1b4e` - Dark base tone
- **Retrowave Orange**: `#ff6c11` - Gradient primary color
- **Laser Pink**: `#ff3864` - Highlight borders
- **Grid Blue**: `#0d0221` - Background grid color

## Typography

- **Headers**: Righteous
- **Body Text**: Varela Round

## Best Used For

Music festivals, trendy fashion, retro gaming, nightlife guides, 80s aesthetic content.
