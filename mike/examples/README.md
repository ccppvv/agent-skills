## 🎯 如何使用案例

### 学习模式1: 跟随式学习
1. 阅读案例背景和问题
2. 跟随每个模型的应用过程
3. 理解模型如何产生洞察
4. 学习模型组合的方法

**适合**: 初学者,想了解模型应用流程

---

### 学习模式2: 对比式学习
1. 先自己尝试分析问题
2. 应用你选择的模型
3. 然后阅读案例
4. 对比你的分析和案例的差异

**适合**: 有一定基础,想提升应用能力

---

### 学习模式3: 迁移式学习
1. 阅读案例
2. 识别关键模式和方法
3. 应用到你自己的类似问题
4. 记录你的应用过程和结果

**适合**: 有经验用户,想解决实际问题

## 💡 案例学习建议

### 第1次阅读: 理解问题
- 专注于问题背景和情境
- 理解决策者的约束和目标
- 思考你会如何处理

### 第2次阅读: 学习方法
- 专注于每个模型的应用过程
- 理解为什么选择这些模型
- 学习模型组合的逻辑

### 第3次阅读: 提取模式
- 识别可复用的分析模式
- 总结关键决策原则
- 思考如何迁移到你的问题

---

## 🎓 练习建议

### 练习1: 复现分析
选择一个案例,尝试自己重新分析:
1. 只看问题背景
2. 选择你的模型
3. 进行分析
4. 对比案例中的分析
5. 反思差异和学习点

### 练习2: 迁移应用
将案例方法应用到你的问题:
1. 识别你的类似问题
2. 参考案例选择模型
3. 应用到你的情境
4. 记录过程和结果
5. 回顾和改进

### 练习3: 创建案例
为你的决策创建案例:
1. 记录问题背景
2. 应用思维模型
3. 记录分析过程
4. 记录最终决策
5. 跟踪实际结果
