## 🎯 30秒决策树 (v4.0)

```
你的问题是...

├─ 🎯 要做决策吗? (该不该/要不要/选哪个)
│  ├─ 重大人生决策 → 第一性原理 + 期望值 + 机会成本 + 可逆性 + 后悔最小化
│  ├─ 商业战略决策 → 第一性原理 + 二阶思维 + 期望值 + 博弈论 + MOAT
│  ├─ 快速可逆决策 → 期望值 + 可逆性 (直接做, 成本低)
│  ├─ 多选一 → 决策矩阵 + 期望值 + 机会成本
│  └─ 方向不明 → VDS 先切方向, 再上模型
│
├─ 🔍 要诊断原因吗? (为什么/根因/怎么回事)
│  ├─ 产品/增长问题 → 系统思维 + 反馈循环 + 激励机制 + 杠杆点
│  ├─ 团队/组织问题 → 系统思维 + 激励机制 + 心理安全 + 确认偏误
│  ├─ 个人效率问题 → 第一性原理 + 帕累托原则 + 瓶颈分析
│  ├─ 复杂系统问题 → 系统思维 + 反馈循环 + 涌现 + 杠杆点
│  └─ 增长失效 → 反馈循环 + 激励机制 + 网络效应 + 二阶思维
│
├─ 🔮 要预测后果吗? (会怎样/如果...那么/趋势)
│  ├─ 短期后果 → 二阶思维 + 期望值
│  ├─ 长期影响 → 二阶思维 + 反馈循环 + 复利效应 + 路径依赖
│  ├─ 风险评估 → 期望值 + 黑天鹅 + 尾部风险 + 压力测试

│  ├─ 组织架构 → 管理幅度 + 康威定律 + 委托 + OKRs
│  ├─ 产品方案 → 第一性原理 + 设计思维 + Jobs to be Done
│  └─ 从零到一 → 第一性原理 + 白痴指数 + 五步工作法 + MVP
│
├─ ⚡ 要优化改进吗? (怎么改进/提效/降低成本)
│  ├─ 系统优化 → 杠杆点 + 瓶颈理论 + 帕累托原则
│  ├─ 成本优化 → 白痴指数 + 边际思维 + 逆向思维
│  ├─ 流程改进 → 瓶颈分析 + 五步工作法 + 反馈循环
│  ├─ 个人提升 → 刻意练习 + 复利效应 + 边际思维
│  └─ 增长加速 → 杠杆点 + 网络效应 + 复利效应 + 边际思维
│
├─ 🤝 要谈判/博弈吗? (怎么说服/利益分配/冲突)
│  ├─ 议价/谈判 → BATNA + ZOPA + 锚定效应 + 互惠
│  ├─ 竞争博弈 → 纳什均衡 + 囚徒困境 + 承诺装置
│  ├─ 冲突解决 → 框架效应 + 积极倾听 + 非零和思维

├─ 📊 要评估方案吗? (值不值/风险收益/回报率)
```

## ⚡ 按时间选择

**5分钟快速**:
- 2-3个模型
- 第一性原理 + 期望值 (覆盖大多数情况)
- 使用 VDS 磨刀5步快速澄清问题

**15分钟标准**:
- 3-5个模型
- 类别多样化 + 交叉校验
- 至少做确认偏误检查

**1小时深度**:
- 5-8个模型
- 五维框架全覆盖 + 反证 + 偏差检查 + P0/P1/P2
- 完整框架-证据-结论对照表

**2小时以上**:
- 五维框架 + VDS 先切方向
- 8-12个模型
- 多视角分析 + 完整文档 + 思维日志

## 📚 核心模型速查

### 🎯 战略模型
**详见**: [strategic.md](../references/strategic.md)

| 模型 | 何时使用 | 一句话 |
|------|---------|--------|
| 第一性原理 | 找根本原因、挑战假设 | 分解到基本真理重建 |
| 二阶思维 | 预测后果 | 思考直接后果之外 |
| 逆向思维 | 避免失败 | 从最坏结果反推 |
| 可逆性 | 降低风险 | 优先可撤销决策 |
| 机会成本 | 比较选择 | 真实成本=放弃的最佳选择 |
| 复利效应 | 长期影响 | 微小改进随时间叠加 |
| 安全边际 | 降低下行 | 留够容错空间 |

### 🧠 认知偏差
**详见**: [cognitive.md](../references/cognitive.md)

| 偏差 | 症状 | 快速检查 |
|------|------|---------|
| 确认偏误 | 只看支持证据 | "什么能反驳这个？" |
| 锚定效应 | 被初始数字影响 | "忽略第一个数字，合理值是多少？" |
| 沉没成本 | 被过去投入绑架 | "如果今天开始，我还会做吗？" |
| 可得性偏差 | 近期事件感觉更可能 | "这真的常见还是只是最近？" |
| 框架效应 | 被表述方式影响 | "换个角度说同样的事实？" |

### 🔄 系统思维
**详见**: [systems.md](../references/systems.md)

| 模型 | 何时使用 | 关键问题 |
|------|---------|---------|
| 系统思维 | 复杂相互关联 | "什么结构导致这个？" |
| 反馈循环 | 自我强化模式 | "什么在强化或平衡？" |
| 杠杆点 | 需要高影响干预 | "最小改变最大效果在哪？" |
| 涌现 | 整体≠部分之和 | "交互产生了什么？" |

### 📊 概率模型
**详见**: [probability.md](../references/probability.md)

| 模型 | 何时使用 | 公式/方法 |
|------|---------|----------|
| 期望值 | 量化不确定结果 | EV = Σ(概率 × 结果) |
| 贝叶斯思维 | 用证据更新信念 | P(A\|B) = P(B\|A)·P(A)/P(B) |
| 基础概率 | 避免忽略统计 | 从人群概率开始 |
| 黑天鹅 | 考虑罕见高影响 | 什么不太可能但灾难性？ |

### 💰 经济模型
**详见**: [economics.md](../references/economics.md)

| 模型 | 何时使用 | 核心洞察 |
|------|---------|---------|
| 激励机制 | 理解行为 | 人们响应激励 |
| 机会成本 | 比较选择 | 成本=放弃的最佳选择 |
| 供需关系 | 理解市场 | 价格在均衡点形成 |
| 边际思维 | 优化决策 | 关注下一单位,非平均 |

## 🔧 常见问题快速解决

| 问题 | 快速解决 |
|------|---------|
| 不知道用哪个模型 | 运行 `python3 scripts/model_matcher.py "问题"` |
| 模型太多了 | 限制到2-3个,设定决策截止时间 |
| 模型结论冲突 | 好事!揭示了权衡。你的价值观决定。 |
| 不会正确应用 | 使用 [analysis-framework.md](../assets/templates/analysis-framework.md) |
| 分析瘫痪 | "足够好" > 完美。现在就决定。 |
| 跨结构问题 | 分两阶段：先诊断再决策 |
