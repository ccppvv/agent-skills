# MVP交付形态模板与案例

## 知识类MVP模板

### PDF指南模板

```
封面：[主题]实战指南
目录（3-5个章节即可）
第1章：你的核心问题是什么（1页）
第2章：为什么现有方案不够好（1页）
第3章：我的方法论/框架（2-3页）
第4章：实操步骤（3-5页，这是核心价值）
第5章：常见问题FAQ（1页）
封底：联系方式/社群入口
```

建议页数：20-40页，不需要更多。
定价参考：19-99元

### 1小时直播模板

```
前5分钟：抛出痛点，引发共鸣
15分钟：讲方法论/框架（不要太多理论）
30分钟：实操演示（这是核心价值）
5分钟：总结要点
5分钟：引导关注/加群
```

建议定价：9.9-49元（初期可以免费，用来验证需求）

### 语音包模板

```
10条语音，每条3-5分钟
每条围绕一个具体问题展开
附一份PDF摘要（1-2页）
```

建议定价：29-69元

## 实物类MVP模板

### 样品预售模板

```
产品描述：1页产品说明
样品数量：10-20份
定价策略：成本价或略低，验证需求为主
交付方式：先收预订，达到数量再制作
```

关键：预售不是骗人，是验证需求的最诚实方式。卖不出去就别做。

### 产品组合包模板

```
3款核心产品组合
每款解决一个具体问题
打包价低于单买总和
附一份使用指南
```

## 服务类MVP模板

### 体验价预约模板

```
服务描述：1段话说明你能帮什么人解决什么问题
体验价：正常价的30%-50%
次数限制：3次体验
交付方式：线上1对1
时长：30-60分钟/次
```

关键：体验价不是降价，是降低用户的决策门槛。

### 咨询服务模板

```
明确你能解决的问题范围（3-5个）
明确你不解决的问题（避免边界模糊）
标准化咨询流程：
  1. 课前问卷（5个问题，了解用户情况）
  2. 咨询对话（聚焦1个核心问题）
  3. 课后总结（3条行动建议）
```
