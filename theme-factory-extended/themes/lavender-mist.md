# Lavender Mist

Lavender mist with dreamy and healing light purple tones.

## Color Palette

- **Lavender**: `#e6e6fa` - Main background color
- **Amethyst**: `#9966cc` - Elegant primary color
- **Mist Grey**: `#dcdcdc` - Neutral transition
- **Deep Violet**: `#4b0082` - Clear text

## Typography

- **Headers**: Dancing Script
- **Body Text**: Raleway

## Best Used For

Aromatherapy, sleep products, romantic literature, artistic creation, wellness content.
