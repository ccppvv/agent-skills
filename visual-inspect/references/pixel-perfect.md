# 像素级精度指南

## 精度标准

| 属性 | 容差 | 测量方法 |
|-----|------|---------|
| 颜色 HEX | 0 | 吸管工具 |
| 字号 px | ±1px | 文本属性 |
| 行高 | ±0.5px | 行高测量 |
| 间距 | ±1px | 标尺工具 |
| 圆角 | ±1px | 边框半径 |

## 颜色提取

```bash
# 使用 macOS DigitalColor Meter
# 或设计工具内置吸管

# 颜色格式转换
HEX → RGB: #0066CC → rgb(0, 102, 204)
RGB → HSL: rgb(0, 102, 204) → hsl(210, 100%, 40%)
```

## 字体识别

| 属性 | 提取方法 |
|-----|---------|
| 字体族 | 设计面板 → Font Family |
| 字号 | Font Size (px/rem) |
| 字重 | Font Weight (400/700) |
| 行高 | Line Height (unitless/px) |
| 字间距 | Letter Spacing (em) |

## 间距系统

```
Box Model:
┌─────────────────────────┐
│        margin           │
│  ┌───────────────────┐  │
│  │     padding       │  │
│  │  ┌─────────────┐  │  │
│  │  │   content   │  │  │
│  │  │             │  │  │
│  │  └─────────────┘  │  │
│  └───────────────────┘  │
└─────────────────────────┘

测量顺序: padding → border → margin
```

## 对齐检查

- 垂直对齐：基线、中线、顶线
- 水平对齐：左、中、右、两端
- 网格对齐：8px/4px 基准网格
